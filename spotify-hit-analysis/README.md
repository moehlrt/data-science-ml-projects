# Spotify Hit Analysis

## Spotify Top Tracks – Data Processing & Analysis
This project processes and analyzes a Spotify Top-50 tracks dataset to understand characteristics of hit songs and answer basic exploration questions.

Google colab link - https://colab.research.google.com/drive/1wlean5KD-9yxhMl2yYh3__6kS0xGC6av?usp=sharing
### Data
- Source: `Dataset/spotifytoptracks.csv` - originally from https://www.kaggle.com/datasets/atillacolak/top-50-spotify-tracks-2020
- Cleaned export: `Dataset/spotifytoptracks_processed.csv`
- Key fields: `artist`, `album`, `track_name`, `genre`, and numeric audio features (e.g., `energy`, `danceability`, `loudness`, `tempo`, `duration_ms`).

### Approach
1. Load data with pandas
2. Data cleaning
   - Normalize casing for text columns
   - Check for missing values and duplicates (rows/columns)
   - Outlier handling: clipping for [0,1] features and IQR-based clipping for numeric features
3. Exploratory analysis
   - Basic stats (row/column counts, categorical vs. numeric)
   - Artist/album/genre frequencies
   - Correlations between audio features (heatmap)
   - Genre comparisons via boxplots (e.g., danceability, loudness, acousticness)
4. Results Overview
5. Interpretation

### Usage
- Main notebook: `Spotify_Dataset.ipynb`
- Run cells in order; processed CSV is generated automatically.
